## Nurse API Endpoints

### 1. Nurse LogIn

- **Method:** POST
- **URL:** `http://localhost:3000/nurse/LogIn`
- **Request Body:**
    ```json
    {
        "fName": "Muhammad Hammad",
        "lName": "Asghar",
        "password": "123"
    }
    ```
- **Response:**
    ```json
    {
        "success": true,
        "nurse": {
            "fName": 15,
            "mI": "Muhammad Hammad",
            "lName": "A",
            "age": "Asghar",
            "gender": 19,
            "phone": "MALE",
            "address": "306-488-9750",
            "password": "SinghPura"
        }
    }
    ```
